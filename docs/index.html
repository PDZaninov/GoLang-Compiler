Hello World

First, here's the classic "Hello World" program.

package main

import "fmt"

func main() { fmt.Println("Hello World!") }

To run the program, put the code into your favorite text editor and save the file as hello-world.go. Then, use the command ./gt hello-world.go. If you would like to see the Go output alongside the Truffle output, use the flag -go.

$ ./gt hello-world.go Hello World!

Values

Go has several types including strings, integers, floats, and booleans.

package main

import "fmt"

func main() { fmt.Println("Hello" + " World!") fmt.Println("4 + 2 = ", 4 + 2) fmt.Println("1.0/3.0 = ", 1.0/3.0) fmt.Println(true) }
